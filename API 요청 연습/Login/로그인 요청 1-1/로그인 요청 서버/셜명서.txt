터미널 에서 이거를 실행해

이상하게 실행 버튼 누르지말고 입력해서 서버 실행 해라

 uvicorn main:app --reload <= 참고로 그냥 이렇게 살행시키지 말고 

uvicorn main:app --host 0.0.0.0 --reload <= 이렇게 실행 시켜라

왜냐하면 이게 -> 0.0.0.0 = 이 컴퓨터가 가진 모든 IP 주소에서 요청을 받겠다 이런 의미야 즉 모든 네트워크에서 접속을 허용한다는 거야

0.0.0.0
→ “누구든 이 포트로 오면 다 받아줄게”

127.0.0.1 <= uvicorn main:app --reload가 기본으로 하는 ip주소야
→ “나 혼자만 들을게”

이 숫자들은 규칙이야 그래서 외우삼 나도 왜 서버로 안보내졌지 싶었는데 이런 거 생각 못해서 고생함

host = 서버가 요청을 받을 IP 주소 <= 대충 이런뜻임 그래서 저 두개의 ip주소는 외워두삼
